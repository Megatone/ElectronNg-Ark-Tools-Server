GameUserSettings.ini

Basic Options


Wide variety of configuration options that control their behavior and adjust many aspects of the game

Custom Levels


This section allows you to set the maximum level and the total experience of the players and dinos

Custom Stats


This section allows you to configure the stats of players and dinos

Custom Engramns


This section allows you to configure the engram points that are delivered by each level

Custom Harvest


This section allows you to set the harvest multiplier of the items

Custom Spawns


This section allows you to generate new spawn of dinos in the official maps
